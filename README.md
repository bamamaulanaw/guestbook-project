# Guestbook Project

A simple guest book (Buku Tamu) web application built using PHP Native and MySQL with SB Admin 2 template.

## Description

This application is designed to manage guest visit data with basic CRUD (Create, Read, Update, Delete) functionality.  
It includes authentication, role-based access (admin and operator), image upload, and Excel report export.

## Tech Stack

- PHP Native
- MySQL
- Bootstrap 4 (SB Admin 2)
- PhpSpreadsheet

## Installation & Usage

1. Clone or download this repository.
2. Place the project folder inside `htdocs`.
3. Create a MySQL database named `app_bukutamu`.
4. Import the database structure according to the project requirements.
5. Configure database connection in `koneksi.php`.
6. Run Apache and MySQL using XAMPP.
7. Access the application via browser:

    ```hash
    http://localhost/buku-tamu
    ```
